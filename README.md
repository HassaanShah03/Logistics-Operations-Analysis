# Logistics-Operations-Analysis
SQL-based analysis of logistics operations to identify inefficiencies in deliveries, drivers, trucks, and facilities. Used metrics like cost per trip/km and delay rates to uncover cost leakages and performance gaps, enabling data-driven decisions for efficiency improvement and cost optimization.
Dataset too large to upload. refer to (https://www.kaggle.com/datasets/yogape/logistics-operations-database/data) for the data.
